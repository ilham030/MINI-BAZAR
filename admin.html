GitHub-a Qoşul

Bu, panelin dəyişiklikləri birbaşa saytınızın reposuna yazması üçün lazımdır. Yalnız bir dəfə, bu cihazda edilir.

Token necə alınır:

  1. github.com saytına daxil olun
  2. Settings → Developer settings → Personal access tokens → Fine-grained tokens
  3. "Generate new token" düyməsinə basın
  4. Repository access → Only select repositories → MINI-BAZAR seçin
  5. Permissions → Repository permissions → Contents → Read and write seçin
  6. "Generate token" basın və çıxan kodu kopyalayın

Token yalnız bu cihazda, brauzerinizdə saxlanılır. Heç kimlə paylaşmayın.
